## 1. 游戏概述

### 一句话介绍
> 双球同步解谜：一个操作，两段路径，同时到达。

### 游戏类型
- 核心类型：益智 / 解谜 (Puzzle)
- 子类型：逻辑 / 对称思维
- 参考竞品：Stephen's Sausage Roll、Baba Is You、Patrick's Parabox

### 目标平台
- 首发：Windows (Steam)
- 后续：macOS, Linux, iOS/Android (移动版)

### 核心卖点
1. **全新解谜机制** — 同时控制两个球，却只能发出同一个指令。这一限制迫使玩家以全新的方式思考路径规划。
2. **极低上手门槛** — 只需方向键，三秒学会规则。但精通需要深度空间推理。
3. **短 session 友好** — 每关耗时 30 秒 ~ 5 分钟，适合碎片时间。
4. **速通潜力** — 每关记录最少步数，满分通关驱动重玩性。
5. **视觉干净优雅** — 暗色主题 + 霓虹光效，在 Steam 众多像素游戏中脱颖而出。

### 定价策略
- 首发价：**$3.99 USD**（约 ¥25）
- 首发折扣：-20% → $3.19
- 完全版（含后续 DLC）：$5.99

### 目标受众
- 解谜游戏爱好者（Baba Is You, The Witness, Portal 玩家）
- 速通社区活跃分子
- 喜欢"轻松但有深度"游戏体验的玩家
- 直播/短视频观众（因机制新颖，观看性强）

## 2. 核心机制

### 基础规则
- 两个球位于两个不同的迷宫网格中，一个青色（A），一个粉色（B）
- 按方向键 → 两个球都朝该方向移动一格
- 如果一个球前方是墙，该球不动，另一个球仍可移动
- 两个球必须**同时**到达各自的目标位置才能过关
- 每关记录步数，步数越少越好

### 进阶机制（后续版本规划）
| 机制 | 说明 | 预计导入关卡 |
|------|------|-------------|
| 冰面 | 进入后沿原方向滑行直到撞墙 | 9-12 |
| 传送门 | 成对出现，进入一个瞬间传送到另一个 | 13-16 |
| 单向门 | 只能从一个方向通过 | 17-20 |
| 压力板 | 踩下后为另一个球打开路径 | 21-24 |
| 分裂路径 | 一个球走左边，另一个走右边 | 25+ |

### 操作方式
- 键盘：WASD / 方向键
- 鼠标：点击画布边缘
- 触摸：滑动（移动端适配）
- 撤销：Ctrl+Z / Cmd+Z

## 3. 技术方案

### 原型技术栈
- HTML5 Canvas + 原生 JavaScript
- 单文件架构，无需构建工具
- 大小：~15KB（压缩后约 6KB）

### Steam 正式版技术栈
```
推荐方案：Godot 4 (C# / GDScript)
├── 优势：免费开源、导出 Windows 原生 exe、轻量(~30MB runtime)
├── Steam API 集成：Steamworks.NET (C#) 或 Godot 原生插件
└── 打包工具：Godot 内置导出器 + Inno Setup (安装程序)

备选方案：Unity 2022 LTS (C#)
├── 优势：生态成熟、文档丰富
├── Steam API 集成：Steamworks.NET
└── 打包工具：Unity Build Pipeline
```

### 游戏体积估算
| 项目 | 大小 |
|------|------|
| 引擎运行时 | ~30-50MB |
| 关卡数据 (~200关) | ~50KB |
| 音效 (~20个) | ~5MB |
| UI 资源 | ~2MB |
| 可执行文件 | ~1MB |
| **总计** | **~40-60MB** |

### 关卡编辑器
- 内置关卡编辑器（开发工具）
- 支持社区创意工坊（Steam Workshop）
- 玩家可设计并分享自定义关卡

## 5. 盈利模式

### 收入结构
```
核心收入 (90%)
├── Steam 游戏销售 — $3.99/份
├── DLC "Split+ 扩展包" — $1.99 (新增 50 关 + 新机制)
└── 完整版捆绑包 — $5.49 (省 $0.49)

辅助收入 (10%)
├── Steam 集换式卡牌 (社区市场交易抽成)
├── 赞助/DLC 合作
└── 移动版广告收入 (如后续推出)
```

### 无内购策略
- 一次性付费，无微交易
- 无氪金、无战令、无抽卡
- Steam 退款政策正常适用（2小时内/14天内）
- 核心盈利靠口碑传播带来的自然销量

### 为什么 $3.99 是合理定价
| 定价 | 心理影响 | 购买转化率 | 推荐度 |

## 6. Steam 发布流程

### 前期准备（发布前 3 个月）
```
Month -3: 完成核心玩法 Demo，申请 Steam 开发者账号 ($100)
Month -2: 创建 Steam 商店页面，准备截图/预告片
Month -1: 参加 Steam Next Fest，收集 wishlist
Month -0.5: 联系评测媒体/主播，发送 Key
Month 0: 正式发布
```

### 所需材料清单
- [ ] Steam 开发者账号（$100 一次性费用）
- [ ] 游戏截图 x5（1280x720，展现不同关卡）
- [ ] 预告片 x1（30-60 秒，展示核心机制）
- [ ] 游戏图标（圆角方形，多种尺寸）
- [ ] 胶囊图（Steam 各种尺寸需求）
- [ ] 游戏描述（中英文）
- [ ] 评测用 Key x20-50
- [ ] 社区讨论区版规

### Steam 页面优化要点
1. **胶囊图（Capsule）**：左右分屏展示两个球 + 迷宫，紫色+青色配色
2. **预告片前 5 秒**：必须展示核心机制——两个球同时移动
3. **短描述**："一款关于同步思考的解谜游戏。控制两个球，发出一个指令。"
4. **标签**：Puzzle、Minimalist、Difficult、Logic、Indie
5. **竞争对手定位**：比 Baba Is You 更易上手，比 Sokoban 更新鲜

## 附录 B：Steam 页面推荐文案

### 短描述
> 双球同步解谜。一个操作，两段路径，同时到达。

### 完整描述
> **双轨 SPLIT** 是一款关于"同步思考"的解谜游戏。
>
> 两个球，两个迷宫，一个指令。当你按下方向键，两个球会同时移动——它们步调一致，却各自走向不同的终点。
>
> 规则简单到一句话就能说明白，但随着迷宫复杂度提升，你会发现需要全新的思维模式来解决这些谜题。你不能分别控制两个球——它们是一体的。
>
> **特色**
> - 80+ 精心设计的手工关卡
> - 5 种独特机制：冰面、传送门、压力板、单向门、分裂路径
> - 内置关卡编辑器 + Steam 创意工坊
> - 每日挑战：同一个种子，与全球玩家比步数
> - 速通模式：争取每关最少步数，冲击排行榜
>
> 一杯奶茶的价格，换来数小时的烧脑乐趣。
